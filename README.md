# 🔐 Gerador de Senhas

Projeto simples de um gerador de senhas desenvolvido com **HTML, CSS e JavaScript puro**.

---

## 🚀 Funcionalidades

- Geração de senhas aleatórias
- Escolha do tamanho da senha
- Opções:
  - Letras
  - Números
  - Símbolos
- Avaliação de força da senha
- Botão para copiar senha

---

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript

---

## 📷 Preview

Interface simples com tema escuro e foco em segurança.

---

## 💡 Como usar

1. Baixe o projeto
2. Abra o arquivo `index.html`
3. Clique em **Gerar senha**
4. Copie a senha gerada

---

## 📌 Objetivo

Projeto criado para praticar lógica de programação, manipulação do DOM e estilização com CSS.

---

## 👨‍💻 Autor

Desenvolvido como exercício de aprendizado front-end.
